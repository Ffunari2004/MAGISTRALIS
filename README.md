# MAGISTRALIS

Magistralis adds a complete set of classical columns, capitals, bases and entablatures so you can build temples, forums, villas, and monuments, in big and small variations, both sizes have the guilded variation, there is also various styles of columns.

## The Three Classical Orders

**Doric:** Plain, fluted shafts with a simple, weighty capital. The oldest and sturdiest of the orders, perfect for foundations, temples, and civic buildings.

**Ionic:** Slender fluted columns topped with the iconic volute (scroll) capital, ideal for libraries, senates, and refined public spaces.

**Corinthian:** The most ornate order, crowned with acanthus-leaf capitals for palaces, grand temples, and imperial architecture.

## Installation

1. Install [Minecraft Forge](https://files.minecraftforge.net/) for Minecraft 1.20.1.
2. Download the latest Magistralis `.jar` from CurseForge.
3. Drop the `.jar` file into your `mods` folder.
4. Launch Minecraft and enjoy!

## Requirements

- Minecraft version: 1.20.1
- Mod loader: Forge
